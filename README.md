# nodejs-microservice-template

A simple NodeJS service template

- Express
- Awilix
- Winston
- Jest

## Getting started
### Requirements
- Node 8
- Npm 5

### Local development

You have to use the following command to start a development server (nodemon):

```sh
npm install
```

```sh
npm run start:dev
```

### Unit tests

```sh
npm test
```
